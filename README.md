# im-safechat 私域聊天系统（前端部分）

这是一个适配移动端 App 的轻量级聊天系统前端页面，配合后端 WebSocket 服务可实现群聊、私聊、验证码登录等功能。

##  页面结构

- `im/index.html`：主聊天窗口页面
- `im/css/style.css`：酒红+米白主题 UI 样式
- `im/js/main.js`：WebSocket 消息发送/接收逻辑
- `im/img/`：聊天 UI 图标资源
- `im/font/`：自定义字体资源（MiSans）

## ✅ 页面特色

- 适配微信内/iframe 使用
- 字体大、按钮清晰，面向中老年用户
- 群聊、私聊统一界面
- App 套壳嵌入，支持安卓启动页
